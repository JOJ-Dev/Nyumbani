# Nyumbani - Rental Management System

## Description

Nyumbani is a system designed to help landlords in managing their rental property easily. Here they can easily track rents and communicate with their tenants easily and reliably.

## Features
- Landlord and Tenant Account Management
- Stripe Payment gateway Integration
- Mpesa-Daraja Payment gateway
- Graphs and Chart Generation from data
- Reciept generation
- Email and message reciepts or reminders
